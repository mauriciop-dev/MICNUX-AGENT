## Persistence & Infrastructure
- **Memory DB:** Supabase (PostgreSQL + pgvector). This is the best option for Micnux to store embeddings of past conversations and architectural decisions.
- **Security:** Use `.env` for all credentials. Never hardcode.
- **A2A Protocol:** Use for cross-platform learning (sharing bug resolutions and user behavior patterns without PII).
